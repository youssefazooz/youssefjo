اقراءه كامله لكي تستطيع أن تتعلم هذا المجال

عليك أن تسأل نفسك هذا السؤال وتفكّر به جيّداً ، اسأل نفسك ماذا تريد أن تصبح ؟ وكم هي المسافة المستعد لسيرها لتصبح “هاكر”؟ اذا كنت تريد تعلّم اختراق الأجهزة والمواقع فقط ليقول الآخرين عنك أنك هاكر أو لأنك تظن أن اختراقك للمواقع سيجعل الآخرين يحترموك ويخافون منك فاعلم أن ما ستقوم به هو مضيعة للوقت! قد تستطيع خلال فترة زمنية قصيرة أن تخترق بعض الأجهزة والمواقع الضعيفة لكن هذا لن يجلب لك الاحترام الذي تبحث عنه، اذا لم تكن ترغب باحتراف مجال الهاكر وتحمّل الأمور المترتبة على ذلك أنصحك ألا تبدأ وألا تضيع وقتك من الأساس. أما اذا كنت تريد أن تصبح هاكر حقيقي أو اخترت الحماية والاختراق كمجال مهني تريد احترافه فيجب أن تعلم أن الطريق الذي اخترته طويل وليس بالبساطة التي يتصوّرها البعض. فبذلك أنت ستحتاج لتعلم واحتراف العديد من الأمور المختلفة بنفس الوقت بدءً من الشبكات ، ادارتها وحمايتها مروراً باحتراف لينكس وأنظمة التشغيل المختلفة انتهاءً بالبرمجة ، اكتشاف الثغرات والهندسة العكسية وقد تصل للهندسة الاجتماعية وأساليب التلاعب بالأشخاص أيضا! الحقيقة لا أحد يستطيع أن يصبح هاكر بين يوم وليلة أو خلال بضعة أيام أو حتى شهور فتعلّم جميع الأمور التي ذكرتها سابقاً ليس بالبساطة التي قد يتصوّرها البعض ويحتاج صبر واصرار كبيرين.

البوت نت تتمثل بشبكة عبارة لصيد الضحاية قام بعملها صاحبها لغرض معين او لعدة اغراض منها : السبام - سرقة الحسابات البنكية - سرقة الايميلات - وحتى اختراق المواقع عن طريق DDos - وعدة اشياء خبيثة غيرها يقوم الشخص بانشاء تلك الشبكة من البوتات. عبر وسائل تساعده وهي برمجية البوت وضيفته استهداف اجهزة مصابة يقسم البوت نت الى نوعيات : 1- : هي بوتات الــ{ HTTP } وهي بوتات التي يكون التبليغ و التحكم فيها عن طريق لوحه تحكم تكون موجودة علي احد السيرفرات ويتم برمجتها باستخدام الدمج ما بين { c++ } و بي اتش بي { php } 2- : وهي بوتات الـ{ irc } وهي بوتات التي كتبت بلغات برمجية مختلفة ويكون التبليغ و التحكم فيها عن طريق جات الـ { irc } ويتم كتابة هذه البوتات الاكثريه منها عن طريق لغات البرمجة { Java } و { C++ } كيفية يتم انشاء الشبكة البوت نت : حينما يقوم الماستر { bot mater }بانشاء ملف يصيب به بعض الاجهزة ويجعلها تتصل تلقائيا بعد اصابتها بوسيط التحكم لديه ووسيط التحكم في هذه الحاله يختلف من بوت الى بوت اخر فهو في بعض الاحيان يكون موقع علي احد السيرفرات وفي الاحوال الاخرى يكون قناة علي جات الـ{ irc } ملاحظة : البوت نت برمجية خبيثة تصيب الاجهزة والكمبيوترات والهواتف ماهي البوتات نت المنتشره والخطيرة : 1- Zeus : شبيه الكيلوجر والاخطر انه يجلب ضحاية تصل الى الاف بليوم 2- MBot : خطير لانه يختفي ولاتكشفه اي حماية 3- rXBot : يعتبر من البوتات الخطيرة والمتداولة بكثرة والمرغوبة 4- Java BoT : وهو بوت خطير لانه لايتم كشفه باي حماية 5- chimera BoT : ويعتبر الاخطر لكنه يتم بيعه او تاجيره لمدة قصيرة فانه يستخدم للجرائم الالكترونية مثال : يمكنك تشبيه البوت نت الخاص بك الذي انت برمجته ك شبكتك الراوتر المنزلية و واي شخص يتصل فيها تستطيع الغاء عنه النت او العب بهاتفه من خلال برامج تحتاج روت .
